# Diabetic Patients Data Analytics Project

## Overview

This project analyzes diabetic patient data to uncover patterns, risk factors, and insights that can support better healthcare decisions. It follows an end-to-end data analytics workflow, including data loading, exploratory data analysis (EDA), data cleaning, SQL-based querying, and interactive dashboard development.

The final outputs include a Power BI dashboard, a detailed report, and a presentation created using Gamma.

The dataset contains medical and demographic information of diabetic patients.
Data is processed and analyzed using Python.

## Tools & Technologies

* **Python** (Pandas, NumPy)
* **MySQL** (for data querying and analysis)
* **Power BI** (for interactive dashboard creation)
* **Gamma** (for presentation/PPT creation)

---

## Project Steps

1. **Data Loading**

   * Imported dataset into Python using Pandas.

2. **Exploratory Data Analysis (EDA)**

   * Analyzed distributions of medical attributes.
   * Identified correlations between features.
   * Visualized trends and patterns.

3. **Data Cleaning**

   * Handled missing or zero values in critical columns.
   * Removed duplicates and outliers.
   * Standardized data formats.

4. **SQL Analysis (MySQL)**

   * Loaded cleaned data into MySQL.
   * Executed queries to extract insights such as:

     * Average health metrics by age group
     * High-risk patient identification

5. **Dashboard Creation (Power BI)**

   * Built a **2-page interactive and interlinked dashboard**.
   * Enabled filtering and drill-down analysis across pages.

6. **Reporting & Presentation**

   * Created a structured report summarizing findings.
   * Developed a professional presentation using Gamma.

## Dashboard

The Power BI dashboard includes:

* **Page 1: Patient Overview Dashboard**
    Understand who the patients are + how hospital operates

* **Page 2: Readmission & Risk Analysis**

  * Risk factor analysis
  * Patient segmentation insights
  * Interactive filters connected across both pages

## How to Run

1. Clone the repository:

   ```bash
   git clone <repository-link>
   ```

2. Install required Python libraries:

   ```bash
   pip install pandas numpy matplotlib seaborn
   ```

3. Run the Jupyter Notebook:

   ```bash
   jupyter notebook
   ```

4. Load the cleaned dataset into MySQL and execute SQL queries.

5. Open the Power BI (.pbix) file to explore the dashboard.

6. Review the report and Gamma presentation for final insights.
